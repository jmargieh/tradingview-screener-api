## Documentation Highlights

### Quick Start Example

The quickstart guide includes a complete working example:

```typescript
import { StockScreener, StockField } from 'tradingview-screener';

const screener = new StockScreener();

screener
  .where(StockField.MARKET_CAPITALIZATION.gt(1_000_000_000))
  .where(StockField.PRICE.between(10, 500))
  .where(StockField.VOLUME.gte(100_000))
  .select(StockField.NAME, StockField.PRICE, StockField.VOLUME)
  .sortBy(StockField.MARKET_CAPITALIZATION, false)
  .setRange(0, 50);

const results = await screener.get();
console.table(results.data);
```
